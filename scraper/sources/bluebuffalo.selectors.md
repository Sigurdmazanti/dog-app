## Selectors

| Field | Selector / Approach |
|---|---|
| Sub-brand title | `h1` (strip `™` and `®` with `/[™®]/g` before use) |
| Variant name | `h3[itemprop=name]` |
| Composed title | cleaned `h1` text + `h3[itemprop=name]` text, joined with a space; falls back to `h3[itemprop=name]` alone when `h1` is absent |
| Guaranteed Analysis tabpanel | `[role=tab]` whose `.text().trim()` equals `"Guaranteed Analysis"`, read `aria-controls`, then select `[id="<value>"]` |
| Analytical constituents | `tr[role=row]` inside the Guaranteed Analysis tabpanel — `th` = nutrient name, `td` = value, formatted as `Name: Value` lines joined with `\n` |
| Ingredients tabpanel | `[role=tab]` whose `.text().trim()` equals `"Ingredients"`, read `aria-controls`, then select `[id="<value>"]` |
| Ingredients description | Direct children of `.u-flex` inside the Ingredients tabpanel — anchor elements: read `data-tag-text` attribute; span elements: read `.text().trim()`; join all in DOM order with `, ` |

## Notes

- URL pattern: `/dog-food/<type>/<slug>/` (e.g. `/dog-food/wet/baby-blue-turkey-potato-recipe/`)
- Tab IDs are generated by PrimeVue (`pv_id_0_tab_N` / `pv_id_0_tabpanel_N`) and must NOT be hardcoded. Use the `aria-controls` attribute of the matching `[role=tab]` button instead.
- Trademark symbols are embedded in a `<sup>` inside `h1`; Cheerio's `.text()` includes them — strip with `/[™®]/g`
- Ingredients mix `<a data-tag-text="...">` tags (ingredients with brand detail pages) and plain `<span>` elements (generic ingredients like "Chicken Broth"). Selecting only anchors will silently drop spans, so iterate all children in DOM order.
- The Guaranteed Analysis header row (`<tr class="sronly">`) contains `<th>` elements without `role=row` and is skipped automatically by the `tr[role=row]` selector.
